# 프로젝트 개요

## 1. 목적
커뮤니티형 백엔드 시스템 구축.

## 2. 기술 스택
- **언어:** Java 21
- **프레임워크:** Spring Boot 3.5.5
- **보안:** Spring Security, JWT (Auth0)
- **데이터베이스:** H2 (개발), MySQL (운영)
- **빌드 도구:** Gradle
- **테스트:** JUnit 5, MockMvc

## 3. 주요 기능
- **사용자 관리:** 회원가입, 로그인, 중복 체크, 유저 조회
- **게시판 관리:** 게시글 작성, 수정, 삭제, 목록 조회, 상세 조회 (N+1 최적화)
- **댓글 관리:** 댓글 작성, 수정, 삭제, 상세 조회
- **권한 관리:** JWT 기반 Stateless 인증, 작성자 본인만 수정/삭제 가능

## 4. 디렉토리 구조 (주요)
- `_core`: 공통 유틸, 설정, 예외 처리
- `auth`: 인증 (JWT 발급)
- `user`: 사용자 도메인
- `board`: 게시글 도메인 (서비스, 리포지토리)
- `reply`: 댓글 도메인
